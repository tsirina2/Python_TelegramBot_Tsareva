# Проект: Telegram-бот с функцией календаря
Этот проект поможет закрепить навыки, которые вы получите по мере прохождения курса. Вы напишете Telegram-бот с функцией календаря, который положите в портфолио. Проект состоит из двух частей. Выполните первую часть в конце модуля «Потоки, процессы и асинхронность», а вторую — в конце модуля «Подготовка к запуску».

<aside>
⚡Выполните первую часть проекта и сдайте ее на проверку код-ревьюеру. Код-ревьюер проверяет каждое решение и пишет подробный отзыв на результаты работы. Откройте ваш репозиторий и добавьте аккаунт `EdusonCodeReview` `фрагмент кода` в коллабораторы. Подробная инструкция по оформлению репозитория есть в описании задания. Проверка работы занимает до 7 рабочих дней.
